# Student Result Management System

A simple Python-based console project to manage student records, calculate grades, and save/load data.

## Features
- Add new students with marks  
- Auto-calculates Total, Average & Grade  
- View all student records  
- Saves data to JSON  
- Loads data automatically on start  

## Tech Stack
- Python  
- File Handling (JSON)  
- Lists & Dictionaries  

##  How to Run
```
python main.py
```
